\# Dockerized Task Manager



\## 📌 Description

A simple task manager web application built using Flask and Docker.



\## 🚀 Features

\- Add tasks

\- Delete tasks

\- Simple UI

\- Containerized using Docker



\## 🛠️ Tech Stack

\- Python (Flask)

\- SQLite

\- Docker



\## ▶️ How to Run

1\. Build image:

&#x20;  docker build -t task-manager .



2\. Run container:

&#x20;  docker run -p 5000:5000 task-manager



3\. Open browser:

&#x20;  http://localhost:5000
